# Frontend

This app uses Vite + React + Tailwind.

Background image:
- Put an image at `public/images/bg.jpg` (or adjust the path in `src/index.css` under `background-image`).
- Recommended to use a compressed image (WebP/JPEG) for performance.

Run locally (from this folder):
- Install deps: `npm install`
- Start dev server: `npm run dev`
